Manual Restore

Restore only while HolyFHIR is closed. These local copies depend on the same local key in .env, so keep the matching .env file with any database you plan to restore.

  1. Close HolyFHIR completely.
  2. Open the app data folder shown below.
  3. Rename the current live database to keep a last-chance copy.
  4. Copy the database copy you want from the folder shown below.
  5. Rename the copied backup to the live database filename.
  6. Reopen HolyFHIR.
