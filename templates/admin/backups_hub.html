Manual Restore

Restore only while HolyFHIR is closed. The backup is encrypted with the same local key in .env, so keep the matching .env file with the restored database.

  1. Close HolyFHIR completely.
  2. Open the app data folder shown below.
  3. Rename the current live database to keep a last-chance copy.
  4. Copy the backup file you want from the backups folder.
  5. Rename the copied backup to the live database filename.
  6. Reopen HolyFHIR.

Then copy a backup into the app data folder and rename it:

holyfhir.encrypted.pre-fhir-import.20260426-120000.sqlite3
holyfhir.encrypted.sqlite3
