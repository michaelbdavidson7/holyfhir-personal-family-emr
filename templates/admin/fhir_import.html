How to download your chart from Epic MyChart

Use this when you want to bring records from a hospital or clinic MyChart account into FamilyChartVault. MyChart screens vary by health system, but the flow is usually similar.

  1. Sign in to the MyChart website for your hospital, clinic, or health system.
  2. Open the menu and look for Sharing Hub, Document Center, Lucy, Download My Record, Download My Data, or Request Medical Records.
  3. Choose the patient, date range, and record categories you want. If there is an option for a complete medical record, health summary, or FHIR/downloadable data, choose that.
  4. Submit the request. Some MyChart accounts create the file immediately; others place it in Requested Records or send a MyChart message when it is ready.
  5. Download the file to your computer. Keep the original ZIP file if MyChart gives you one.
  6. Come back here, choose the patient in FamilyChartVault, select the ZIP or JSON file, and click Import FHIR data.
If you only see a PDF visit summary, you can still keep it as a document, but this importer works best with the MyChart ZIP, FHIR JSON, or NDJSON export.

FAQ

What file should I upload?
A MyChart requested-record ZIP is best. This page also accepts NDJSON or pasted FHIR JSON.
What if I cannot find the download page?
Search MyChart for "download", "record", "Lucy", or "Sharing Hub". If your health system hides the option, ask their medical records office for an electronic copy of your records in FHIR or downloadable patient access format.
Should I unzip the file first?
No. Upload the ZIP directly when MyChart gives you one.
Is this the same as sending records to another doctor?
No. This is for downloading a personal copy so you can import it into FamilyChartVault.
Do I need to create a patient in this app first?
No, importing will create the patient for you. Alternatively you can attach your file to an existing patient in this app.
Will my records update in this app automatically?
No. At this time, this importer is a one-time upload. If you want to update your records, you will need to download a new file from MyChart and import it again, but it will not overwrite existing data.
